# dice-roller-3d

A polished mobile-friendly 3D dice roller that shows a tumbling die and lands on a random number when tapped.

## Features

- Real-time 3D die rendered with Three.js
- Tap or click to roll
- Smooth tumbling animation with easing
- Random result from 1 to 6
- Responsive layout for mobile and desktop
- Premium glassmorphism UI

## Files

- `index.html` – app entry
- `styles.css` – styling and layout
- `app.js` – Three.js scene, die creation, animation, and roll logic

Open `index.html` in preview to use it.
